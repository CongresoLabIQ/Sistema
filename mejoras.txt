### Tarea 10: Corregir sort sesgado en selectRandom

**Archivo:** `js/evaluation-assignment.js:32`

- Reemplazar `arr.sort(() => 0.5 - Math.random())` con Fisher-Yates shuffle:

```javascript
selectRandom(arr, n) {
  const shuffled = [...arr];
  for (let i = shuffled.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [shuffled[i], shuffled[j]] = [shuffled[j], shuffled[i]];
  }
  return shuffled.slice(0, n);
}
Criterio de éxito: La selección aleatoria no tiene sesgo estadístico.
